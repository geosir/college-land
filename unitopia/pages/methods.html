Methods

Work on this project was fundamentally a treasure hunt, both for data as well as historical references on observations in the data. It involved gathering digital shape data for rendering the map; aggregating this data with property assessment data, which was provided separately; processing and stylizing the data; and finally compiling it into map image tiles for fast delivery. Once the Unitopia map was complete, the work of research involved discovering interesting properties and regions on the map and looking for library or online references which might explain those instances.

Gathering the Data

I began with Cambridge. First I obtained a GeoJSON file describing parcel boundaries from the Cambridge GIS Dataset GitHub. These did not have property data, so I wrote a script to scrape property data from the Cambridge Property Database. This took a long time to run (about a 1.5 days), and in hindsight, the data was also available as a CSV file in the same folder of the GitHub repository. The result of the script was one large GeoJSON file with populated assessment fields for each parcel shape.

Next was Boston, which sported a very similar process. I found the GeoJSON file for the Boston parcels from the Boston Open Data Portal. Then, after corresponding with the City of Boston Assessing Department, I was directed to an online record of their city assessment data. Finally, I acquired a list of human-readable names for property occupancy codes used in the assessment data in order to decode land use. Once again, I wrote a script to combine all of these into one large GeoJSON file with populated assessment fields for each parcel shape.

Working the Data

To better visualize the data as I worked with it (and it do it efficiently, as all these shapes were slowing my browser), I uploaded the data to ArcGIS, which we have a free license for as Harvard affiliates. There, I was able to color the data according to the LBCS standard.

This was also the stage in which I created the layers highlighting Harvard- and MIT-owned properties. This was accomplished by inspecting the recorded owner names of known Harvard and MIT properties and using those names to filter all of the properties by owner. This worked quite well and produced what looks to be most of the properties; however, especially in the case of Harvard, many more owned properties might be listed under other owner names that may be subsidiaries and can be quite obfuscated. Furthermore, the Boston ownership data was massively inconsistent, often listing the same owner in many different formats, truncations, and abbreviations, which was difficult to work with and may have resulted in missed data.

Serving the Data

After processing, the datasets for this project were on the order of several hundred megabytes; plus, displaying thousands of individual polygons representing properties was difficult for my browser. I needed to find a way to serve the data efficiently and lazilyâ€”that is, only load the data that was needed, and in a format that did not require drawing individual polygons.

The solution was to use ArcGIS services to compile the colored data layers into image map tiles. These loaded much faster as the map would only need tens of these tiles at any time, and each tile would represent a drawing of hundreds of shapes at once; plus, displaying images is much easier than drawing individual polygons.

Unfortunately, transparency information is lost on the current implementation of map tiles in Unitopia (which is why sometimes the streets are obscured by the land-usage colorings). For this reason, I did not compile the university property outlines to tiles lest they should obscure the coloring beneath them.

Reference Roundup

Once I had a good visualization, I looked around the map for properties in interesting places, as well as properties that surprised me to be owned by Harvard and MIT. I visited some of these properties on foot, such as Shaler Lane and the Co-Op. Then, I got to work at the Francis Loeb Library, looking for Harvard planning documents and maps that could give background on these interesting properties. The information I learned from these and other readings became the explanatory essays included with Unitopia.
